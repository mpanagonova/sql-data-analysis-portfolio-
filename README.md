# SQL Data Analysis Portfolio

This repository contains SQL queries solving real-world business scenarios using core and advanced SQL concepts.

> The queries are written with readability and practical use in mind.

---

## 📂 Structure

- **aggregate-functions/** – data aggregation and reporting (COUNT, SUM, AVG, GROUP BY, HAVING)
- **scalar-functions/** – data formatting and transformation (string & date functions, CASE)
- **subqueries-and-ctes/** – advanced filtering and multi-step analysis using subqueries and CTEs
- **window-functions/** – advanced analytics, ranking, and time-based calculations

---

## 🔍 Skills Demonstrated

- Data aggregation and summarization
- Data transformation and cleaning
- Business logic implementation using CASE
- Working with relational data (JOINs)
- Subqueries and Common Table Expressions (CTEs)
- Window functions for advanced analysis

---

This project demonstrates the ability to:
